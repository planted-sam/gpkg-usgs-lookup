# gpkg-usgs-lookup

Simple program for searching local GeoPackage files for specific polygons meeting an intersection criteria (WIP)

## setup (WIP)

1. `docker run build . -t test-usgs-lookup`
2. `docker run -it test-usgs-lookup`
